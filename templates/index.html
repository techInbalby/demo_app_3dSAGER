About 3dSAGER

The 3dSAGER (3D Spatial-Aware Geospatial Entity Resolution) pipeline represents a breakthrough in geospatial entity resolution for 3D objects. Our system combines novel geometric featurization with coordinate-agnostic matching to deliver unprecedented accuracy in identifying matching spatial objects across different datasets.

This demonstration showcases the core capabilities of our research pipeline, including 3D mesh analysis, geometric property extraction, and BKAFI blocking. The system is designed to handle complex real-world urban scenarios with high fidelity and computational efficiency, particularly in disaster management and urban planning.

Technology Stack

  • Backend: Flask, Python, NumPy, Scientific Computing
  • 3D Visualization: CesiumJS, Three.js, WebGL, CityJSON rendering
  • Data Display: Pre-computed results visualization (XGBoost predictions, BKAFI blocking results)
  • Web Framework: Flask, Modern JavaScript, Responsive Design
  • Visualization: Advanced 3D rendering, Interactive building comparison
  • Performance: Optimized file loading, Caching for fast lookups

About 3dSAGER

This demo application visualizes results from the 3dSAGER (3D Spatial Attribute-based Geospatial Entity Resolution) pipeline.

3dSAGER is a research project for matching and resolving entities between different 3D city model datasets using coordinate-agnostic geometric features and machine learning.

Repository: 3dSAGER Repository

Paper: 3dSAGER Technical Report

Authors: Bar Genossar, Sagi Dalyot, Roee Shraga, Avigdor Gal

This application is intended for academic and research purposes only.
